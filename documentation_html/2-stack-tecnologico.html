2.1. Tecnologías Principales:

  • HTML5 - Estructura y semántica web
  • CSS3 - Estilos y diseño responsive
  • JavaScript ES6 - Lógica de aplicación y manipulación del DOM
  • Service Workers - Funcionalidad offline y caching
  • PWA (Progressive Web App) - Experiencia similar a aplicación nativa

2.2. Dependencias Externas Clave:

  • Geolocation API - Obtención de coordenadas del usuario
  • Fetch API - Consumo de servicios meteorológicos externos
  • APIs de servicios meteorológicos externos (configuradas en config.js)

2.3. Herramientas de Desarrollo Sugeridas:

  • Visual Studio Code (evidenciado por directorio .vscode/)
  • Navegadores modernos con soporte para PWA
  • Herramientas de desarrollo para testing de Service Workers
  • Lighthouse para auditoría de PWA
