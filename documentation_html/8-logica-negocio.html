8.1. Flujos de Trabajo Importantes:

  1. Obtención de ubicación del usuario
  2. Consulta a API meteorológica
  3. Presentación de datos con opción de unidad de temperatura
  4. Cache de recursos para funcionamiento offline

8.2. Componentes Clave Involucrados:

  • Geolocation API
  • Service Worker
  • Módulo de API (api.js)
  • Módulo de Pronóstico (forecast.js)
  • Módulo de Ubicación (location.js)

8.3. Explicación del Patrón MVC:

Controladores (C):

app.js coordina la interacción entre los módulos

Modelos (M):

api.js maneja los datos de la API externa

Vistas/Componentes Frontend (V):

HTML/CSS para la presentación de datos
