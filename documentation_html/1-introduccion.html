1.1. Nombre del Proyecto:

Weather App

1.2. Descripción General:

Aplicación web progresiva (PWA) que muestra el pronóstico del tiempo basado en la ubicación del usuario. Permite alternar entre grados Celsius y Fahrenheit.

1.3. Visión General del Negocio/Dominio:

Aplicación meteorológica diseñada para proporcionar información climática precisa y accesible con soporte offline mediante Service Workers.

1.4. Audiencia:

Usuarios finales que necesitan información meteorológica actualizada, con especial enfoque en dispositivos móviles.
