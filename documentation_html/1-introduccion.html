1.1. Nombre del Proyecto:

Weather App

1.2. Descripción General:

Aplicación web progresiva (PWA) que muestra el pronóstico del tiempo basado en la ubicación del usuario. Permite alternar entre grados Celsius y Fahrenheit.

1.3. Visión General del Negocio/Dominio:

Aplicación de consumo para usuarios que necesitan información meteorológica precisa y accesible desde cualquier dispositivo.

1.4. Audiencia:

Usuarios finales que necesitan consultar el pronóstico del tiempo. Desarrolladores que quieran entender o contribuir al proyecto.
