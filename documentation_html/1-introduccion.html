1.1. Nombre del Proyecto:

Weather App

1.2. Descripción General:

Aplicación web progresiva (PWA) que muestra el pronóstico del tiempo basado en la ubicación del usuario, con soporte para múltiples unidades de temperatura (Celsius/Fahrenheit).

1.3. Visión General del Negocio/Dominio:

Aplicación meteorológica diseñada para proporcionar información climática precisa y accesible a usuarios finales.

1.4. Audiencia:

Usuarios generales que necesitan información meteorológica actualizada, accesible desde navegadores web y como aplicación instalable (PWA).
