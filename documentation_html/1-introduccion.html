1.1. Nombre del Proyecto:

Weather App

1.2. Descripción General:

Aplicación web progresiva (PWA) para consultar información meteorológica. La aplicación utiliza la geolocalización del usuario y APIs externas para proporcionar datos climáticos en tiempo real y pronósticos.

1.3. Visión General del Negocio/Dominio:

Aplicación de consumo que proporciona servicios de información meteorológica a usuarios finales. El dominio principal se centra en la presentación de datos climáticos de manera accesible y responsive.

1.4. Audiencia:

Usuarios finales que necesitan consultar información meteorológica desde dispositivos móviles y desktop. La aplicación está diseñada para ser accesible y fácil de usar para el público general.
