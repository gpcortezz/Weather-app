1.1. Nombre del Proyecto:

Weather App

1.2. Descripción General:

Aplicación web progresiva (PWA) que muestra el pronóstico del tiempo basado en la ubicación geográfica del usuario. La aplicación funciona offline gracias al uso de Service Workers y ofrece una interfaz responsive adaptada a diferentes dispositivos.

1.3. Visión General del Negocio/Dominio:

La aplicación está diseñada para usuarios que necesitan acceso rápido y confiable al pronóstico del tiempo, incluso en condiciones de conectividad limitada. No tiene fines comerciales directos, sino que sirve como demostración de tecnologías web modernas.

1.4. Audiencia:

Usuarios finales que necesitan información meteorológica y desarrolladores interesados en aprender sobre PW
