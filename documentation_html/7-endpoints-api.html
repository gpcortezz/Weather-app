7.1. Visión General de la API:

La aplicación utiliza la API de AccuWeather para obtener datos de ubicación y pronóstico del tiempo.

7.2. Lista de Endpoints:

  • Obtener ubicación por coordenadas
  • Obtener pronóstico por locationKey

7.3. Detalle de Endpoints Clave:

// Obtener ubicación
${config.api.baseUrl}${config.endpoints.location}?apikey=${config.api.key}&q=${lat},${lon}

// Obtener pronóstico
${config.api.baseUrl}${config.endpoints.forecast}/${locationKey}?apikey=${config.api.key}&metric=${isCelsius}
