4.2. Archivos Clave:

  • index.html: Punto de entrada principal de la aplicación
  • manifest.json: Configuración de la PWA
  • sw.js: Service Worker para funcionalidad offline
  • scripts/app.js: Lógica principal de la aplicación
  • scripts/api.js: Comunicación con la API meteorológica
  • styles/styles.css: Estilos principales de la aplicación
