  • Soporte para Geolocation API
  • Service Workers habilitados

5.3. Pasos de Instalación (Entorno de Desarrollo):

  1. Clonar el repositorio: git clone https://github.com/gpcortezz/Weather-app.git
  2. Abrir el directorio del proyecto en Visual Studio Code
  3. Configurar las APIs externas en archivo config.js según documentación
  4. Abrir index.html en navegador local o usar servidor local (ej: Live Server)

5.4. Pasos de Despliegue (Entorno de Producción):

  1. Configurar servidor web (Apache, Nginx, o servicio de hosting estático)
  2. Subir todos los archivos al servidor
  3. Configurar HTTPS (requerido para Service Workers y Geolocation API)
  4. Verificar que el archivo config.js tenga las configuraciones correctas de producción
  5. Registrar el Service Worker para funcionalidad PWA
